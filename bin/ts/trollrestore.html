$ TrollRestore - метод установки TrollStore через восстановление модифицированной резервной копии.

Выбирайте версию TrollRestore строго для вашей ОС: TrollRestore.exe для Windows, TrollRestore_Linux.zip для Linux, TrollRestore_macOS_amd64.zip для MacOS (процессор Intel), TrollRestore_macOS_armd64.zip для MacOS (процессор Apple Silicon).

guest@otherside ~ % требования.sh

$ Необходимые компоненты:

  • [tool] > python_3
  • [tool] > itunes (для windows)
  • [tool] > trollrestore
  • [app] > советы (из app store)
Для macOS / Linux распакуйте скачанный zip-архив.
Перед началом обязательно отключите "Найти iPhone / iPad" в настройках устройства. После завершения установки TrollStore можно снова включить.

guest@otherside ~ % методы_установки.sh

$ Выберите способ установки:

  • [link] > через_macos_или_windows
  • [link] > через_linux

guest@otherside ~ % способ_через_macos_или_windows.sh

1. Подключите устройство к компьютеру

2. Разрешите доступ к устройству при запросе

3. Откройте папку с TrollRestore

4. Запустите TrollRestore (двойной клик)

5. Введите Tips для замены стандартного приложения "Советы" на персистентный помощник

6. Нажмите Enter и дождитесь перезагрузки устройства

guest@otherside ~ % способ_через_linux.sh

1. Откройте терминал

2. Перейдите в папку с TrollRestore (представленный путь к папке может отличаться):

cd ~/Downloads/TrollRestore_Linux

3. Установите зависимости:

pip3 install -r requirements.txt

4. Подключите устройство к компьютеру

5. Запустите TrollRestore:

python3 trollstore.py

6. Введите Tips для замены стандартного приложения "Советы" на персистентный помощник

7. Нажмите Enter и дождитесь перезагрузки устройства

guest@otherside ~ % установка_trollstore.sh

1. Откройте приложение Tips (Советы)

2. Нажмите "Install TrollStore"

3. Дождитесь респринга устройства

guest@otherside ~ % повторное_внедрение_помощника.sh

1. Откройте TrollStore на домашнем экране

2. Перейдите в Settings → Persistence → Install Persistence Helper

3. Выберите Tips (Советы) из списка приложений

guest@otherside ~ % завершение.sh

$ TrollStore успешно установлен!

$ Для установки IPA-файлов используйте кнопку "+" в TrollStore.
