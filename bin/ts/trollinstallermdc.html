$ TrollInstallerMDC - утилита, использующая уязвимость MDC (MacDirtyCow) для установки TrollStore.

guest@otherside ~ % подготовка.sh

$ Необходимые компоненты:

  • [file] > trollinstallermdc.ipa
  • [tool] > sideloadly
  • [tool] > itunes (для windows)
  • [app] > советы (из app store)

guest@otherside ~ % установка_trollmisaka.sh

1. Скачайте и установите Sideloadly

2. Для Windows установите последнюю версию iTunes

3. Подключите iOS / iPadOS устройство к компьютеру

4. Разрешите доступ к устройству при запросе

5. Откройте Sideloadly и перетащите TrollMisaka-v2.0.ipa в окно программы

6. Введите Apple ID, который в настоящий момент используется на вашем устройстве

7. Нажмите "Start" и дождитесь завершения установки

8. После установки перейдите в Настройки → Основные → Управление устройством

9. Доверьте сертификат, относящийся к вашему Apple ID

guest@otherside ~ % внедрение_помощника.sh

$ Если приложение Tips (Советы) было открыто - перезагрузите устройство.

1. Откройте TrollInstallerMDC

2. Дождитесь сообщения "Successfully replaced Tips binary"

3. Закройте TrollInstallerMDC

$ При ошибке - закройте и снова откройте TrollInstallerMDC.

guest@otherside ~ % установка_trollstore.sh

1. Откройте приложение Tips (Советы)

2. Нажмите "Install TrollStore"

3. Дождитесь респринга устройства

guest@otherside ~ % повторное_внедрение_помощника.sh

1. Откройте TrollStore

2. Перейдите в Settings → Persistence → Install Persistence Helper

3. Выберите Tips из списка приложений

guest@otherside ~ % завершение.sh

$ TrollStore успешно установлен!

$ Для установки IPA-файлов используйте кнопку "+" в TrollStore.
