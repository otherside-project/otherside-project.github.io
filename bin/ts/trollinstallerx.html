guest@otherside ~ % способ_через_компьютер.sh

$ Необходимые компоненты:

  • [file] > trollinstallerx.ipa
  • [tool] > sideloadly
  • [tool] > itunes (для windows)
  • [app] > советы (из app store)

1. Скачайте и установите Sideloadly

2. Для Windows установите последнюю версию iTunes

3. Подключите iOS / iPadOS устройство к компьютеру

4. Разрешите доступ к устройству при запросе

5. Откройте Sideloadly и перетащите TrollInstallerX.ipa в окно программы

6. Введите Apple ID, который в настоящий момент используется на вашем устройстве

7. Нажмите "Start" и дождитесь завершения установки

8. После установки перейдите в Настройки → Основные → Управление устройством

9. Доверьте сертификат, относящийся к вашему Apple ID

Настоятельно рекомендуется включить режим разработчика (Настройки → Конфиденциальность → Режим разработчика).

guest@otherside ~ % способ_без_компьютера.sh

$ Необходимые компоненты:

  • [app] > советы (из app store)

1. Откройте Safari на iOS / iPadOS устройстве

2. Перейдите на jailbreaks.app

3. Найдите TrollInstallerX в списке приложений

4. Нажмите "Install" и дождитесь загрузки

5. Перейдите в Настройки → Основные → Управление устройством

6. Найдите и доверьте сертификат разработчика

Способ с установкой через сайт jailbreaks.app может не работать из-за блокировки корпоративных сертификатов.
На iOS / iPadOS 16+ после установки необходимо включить Режим разработчика в Настройки → Конфиденциальность → Режим разработчика.

guest@otherside ~ % установка_trollstore.sh

1. Откройте TrollInstallerX на устройстве

2. Нажмите "Install TrollStore"

3. Выберите Tips (Советы) в качестве персистентного помощника

$ При зависании на "Exploiting Kernel" перезагрузите устройство.

Выбранное приложение будет полностью заменено помощником - рекомендуем выбрать Tips (Советы).

guest@otherside ~ % примечания.sh

$ Если замена системного приложения на помощник не произошла, то перейдите в TrollStore → Settings → Persistence → Install Persistence Helper → Tips

$ Если TrollStore не установился: откройте приложение-помощник (Tips) → "Install TrollStore"

guest@otherside ~ % завершение.sh
