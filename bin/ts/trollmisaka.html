$ Необходимые компоненты:

  • [file] > trollmisaka.ipa
  • [tool] > sideloadly
  • [tool] > itunes (для windows)
  • [app] > советы (из app store)

guest@otherside ~ % установка_trollinstallermdc.sh

1. Скачайте и установите Sideloadly

2. Для Windows установите последнюю версию iTunes

3. Подключите iOS / iPadOS устройство к компьютеру

4. Разрешите доступ к устройству при запросе

5. Откройте Sideloadly и перетащите TrollInstallerX.ipa в окно программы

6. Введите Apple ID, который в настоящий момент используется на вашем устройстве

7. Нажмите "Start" и дождитесь завершения установки

8. После установки перейдите в Настройки → Основные → Управление устройством

9. Доверьте сертификат, относящийся к вашему Apple ID

guest@otherside ~ % настройка_клавиатуры.sh

1. Откройте приложение TrollMisaka (оставьте его работать в фоне)

2. Перейдите в Настройки → Основные → Клавиатура → Клавиатуры

3. Нажмите "Новые клавиатуры" и выберите TrollMisaka

4. Вернитесь назад и нажмите на TrollMisaka в списке клавиатур

5. Включите "Полный доступ" и подтвердите

6. Перезагрузите устройство (не закрывая TrollMisaka!)

Ни в коем случае не закрывайте TrollMisaka полностью до окончания установки! Просто сверните его.

guest@otherside ~ % активация_помощника.sh

1. Разблокируйте устройство после перезагрузки

2. Свайпните вниз по центру экрана для вызова поиска Spotlight

3. Нажмите на значок глобуса для выбора клавиатуры

4. Выберите TrollMisaka из списка клавиатур

5. Нажмите "kopen"

6. На появившейся кнопке нажмите "Install TrollStore"

$ Если устройство перезагрузится - повторите шаги 1-6

guest@otherside ~ % финальная_установка.sh

1. Откройте TrollMisaka (обязательно из рабочего стола или библиотеки приложений, не закрывая уже открытое окно)

2. Нажмите "Install TrollStore" и дождитесь респринга

guest@otherside ~ % внедрение_помощника.sh

1. Откройте TrollStore

2. Перейдите в Settings → Persistence → Install Persistence Helper

3. Выберите Tips из списка приложений

guest@otherside ~ % завершение.sh

$ TrollStore успешно установлен!

$ Для установки IPA-файлов используйте кнопку "+" в TrollStore.
