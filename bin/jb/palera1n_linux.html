$ Руководство по установке palera1n на компьютере с операционной системой на базе Linux.

На Ubuntu и других Debian-подобных дистрибутивах могут возникнуть проблемы. Рекомендуется использовать другой дистрибутив или palen1x для windows. Также на компьютерах с процессорами AMD Ryzen возможны проблемы, поэтому лучше использовать Mac или ПК с Intel CPU.

guest@otherside ~ % подготовка.sh

1. Откройте терминал

2. Остановите usbmuxd:

sudo systemctl stop usbmuxd

3. Запустите usbmuxd вручную:

sudo usbmuxd -f -p

4. Откройте второй терминал для следующих команд

guest@otherside ~ % установка_palera1n.sh

1. Установите palera1n:

sudo /bin/sh -c "$(curl -fsSL https://static.palera.in/scripts/install.sh)"

2. Введите пароль администратора при запросе

guest@otherside ~ % активация_джейлбрейка.sh

$ Для кабелей USB-C/Lightning могут быть проблемы с входом в DFU. Лучше использовать USB-A/Lightning.

1. Выполните команду:

sudo palera1n -l

2. Подключите устройство и нажмите Enter

3. Следуйте инструкциям на экране для входа в DFU режим

guest@otherside ~ % решение_проблем.sh

$ Если устройство с процессором A9 или старше зависло в pongoOS:

1. Нажмите Control+C

2. Повторите команду:

sudo palera1n -l

$ Это нужно делать при каждом повторном джейлбрейке.

guest@otherside ~ % завершение.sh

$ После перезагрузки устройства откройте приложение palera1n loader и установите Sileo.

$ Для повторного джейлбрейка просто вновь выполните команду:

sudo palera1n -l

$ На этом установка джейлбрейка palera1n завершена!

$ Теперь вы можете устанавливать твики через Sileo.
