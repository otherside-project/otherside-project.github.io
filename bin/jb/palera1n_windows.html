$ Руководство по установке palera1n на компьютере с операционной системой Windows.

$ palen1x - загрузочная среда Linux для запуска palera1n на Windows-компьютерах.

guest@otherside ~ % подготовка.sh

$ Скачайте необходимые файлы по ссылкам:

  • [file] > образ_palen1x
  • [tool] > ventoy

guest@otherside ~ % создание_загрузочной_флешки.sh

1. Запустите Ventoy2Disk.exe

2. Выберите ваш USB-накопитель

3. Нажмите "Install" (все данные с USB-накопителя будут удалены!)

4. Скопируйте palen1x.iso на USB-накопитель

guest@otherside ~ % загрузка_palen1x.sh

1. Перезагрузите компьютер

2. Войдите в BIOS (нажимайте F2 или Del при запуске компьютера)

3. Отключите Secure Boot

4. Выберите загрузку с USB-накопителя

5. На экране Ventoy нажмите Enter

6. В меню palen1x выберите "Shell"

guest@otherside ~ % активация_джейлбрейка.sh

$ Для кабелей USB-C/Lightning могут быть проблемы с входом в DFU. Лучше использовать USB-A/Lightning.

1. Выполните команду:

palera1n -l

2. Подключите устройство и нажмите Enter

3. Следуйте инструкциям на экране для входа в DFU режим

guest@otherside ~ % решение_проблем.sh

$ Если устройство с процессором A9 и старше зависло в pongoOS:

1. Нажмите Control+C

2. Вновь выполните команду:

palera1n -l

$ Эту процедуру нужно выполнять при каждом повторном джейлбрейке.

guest@otherside ~ % завершение.sh

$ После перезагрузки устройства откройте приложение palera1n loader и установите Sileo.

$ Для повторного джейлбрейка просто вновь загрузитесь с флешки в palen1x и выполните команду:

palera1n -l

$ На этом установка джейлбрейка palera1n завершена!

$ Теперь вы можете устанавливать твики через Sileo.
